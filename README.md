## ロードバイクを自慢できるアプリです。

### SNS 等でロードバイクの写真を見てこんなことを思ったことはありませんか？

- なんてメーカーだろう？
- このコンポかっこいいけどどこのだろう？
- ホイールのブランド知りたい
- 同じフォーマットで色々なバイクを見たい

### こんな疑問を解決できるアプリになってます

- 自分の自慢のロードバイクを自慢できる
- 他の人のロードバイクがどんな構成か知ることができる
- DM を送ることもできる
- Twitter にスクショして投稿することもできる
